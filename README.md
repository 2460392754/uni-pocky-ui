# uni-pocky-ui

## Sass to Css

```bash
$ npm run sass
```

## Sass to Css watch

```bash
$ npm run sass:watch
```

## upx to px

2upx => 1px <br />
./dist/styles/index.css => ./dist/styles/index.px.css

```bash
$ npm run upxToPx
```

## upx to px watch

```bash
$ npm run upxToPx:watch
```

## lint

```bash
$ npm run lint
```

## format

```bash
$ npm run bash
```

## docs dev

```bash
$ npm run docs:dev
```

## docs build
